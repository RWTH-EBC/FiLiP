Examples

This directory contains examples to support you how to use FiLiP to build smart solutions based on FIWARE.
 In order to execute the examples you need a working FIWARE instance. For more details look into the required prerequisites of the library.
 Each example should be executable with only small adjustments according to your FIWARE instance, e.g. URL settings. To do so, you can create a copy of the .env.filip.EXAMPLE file in the root folder (i.e., ``…/FiLiP``) and name it .env.filip. By default, we assume that the FIWARE instance is running on localhost.
